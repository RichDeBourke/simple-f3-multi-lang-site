배우기

Bootstrap 4 및 Fat-Free Framework에 대한 경험을 얻기 위해이 사이트를 만들었습니다.

Bootstrap 4

부트 스트랩은 반응 형 웹 사이트를 구축하기위한 널리 사용되는 프론트 엔드 프레임 워크입니다.

Fat-Free Framework

Fat-Free는 상위 10 개 PHP 프레임 워크에서 선정되었습니다.

개요

WordPress 또는 기타 CMS (Content Management System)를 사용하여 전문적인 웹 사이트를 쉽게 만들 수 있지만 사이트를 주식 테마와 조금 다르게 보이게하려면 디자인을 사용자 정의해야합니다. 그것은 복잡해질 수 있습니다. (WordPress 테마를 수정하는 것보다 직선 HTML 코드를 수정하는 것이 더 쉽습니다.)

전자 상거래 사이트를 만들거나 사용자가 로그인해야하는 경우 원하는 보안 기능이 포함되어있어 WordPress를 사용하는 것이 좋습니다.

그러나 사용자와 가장 복잡한 상호 작용이 필요한 연락처 양식이고 공유 템플릿, 깨끗한 URL, 자동 언어 선택 및 사이트 디자인의 융통성으로 구성된 페이지를 원한다면 여기에 설명 된 설정이 좋은 해결책이 될 수 있습니다.

세부 섹션에 사이트를 설정하는 방법에 대한 설명을 제공했으며 모든 파일은 GitHub에 있습니다.

Bootstrap

Bootstrap 은 다양한 디스플레이 크기로 조정되는 반응 형 그리드를 제공하며, 인기가 Bootstrap 및 GitHub에서 많은 수의 코드 스 니펫을 사용할 수 있습니다. 또한 Google을 통해 사용할 수있는 부트 스트랩 사용에 대한 많은 정보가 있습니다.

Fat-Free

Fat-Free 가 가장 인기있는 PHP 프레임 워크 (Laravel 는 아니지만), 공유 호스팅 계획 (Laravel doesn’t 은 쉽지 않음 ) 에서 작동 하며 Fat-Free는 빠른 학습 곡선으로 간주됩니다.

내가 사용하고있는 주요 Fat-Free 구성 요소는 다음과 같습니다.

  • 라우팅 엔진 – 사용자의 요청을 적절한 제어 기능으로 라우팅 합니다.
  • 템플릿 엔진 – HTML 템플릿을 Fat-Free의 "hive"에 저장된 데이터와 결합합니다. 변수는 키 / 값 쌍 형태의 변수를 메모리에 저장하여 웹 페이지를 구성합니다.
  • 다국어 지원 – Fat-Free는 키 / 값 쌍을 사용하여 사전 조회 기능을 통해 여러 언어를 지원합니다. 각 언어의 사전 파일은 dict 디렉토리에 있으며 Fat-Free는 템플릿의 토큰 및 LANGUAGE 설정에 따라 사전 파일에서 원하는 언어 문자열을 선택합니다.
    • 모든 페이지에서 동일한 내용 (예 : Scroll-to-top 버튼) 또는 모든 언어에서 공통적이고 제한된 텍스트 (예 : 연락처 페이지)가있는 페이지에 대해서는 무 지방 언어 지원을 사용합니다.
  • Language detection – Fat-Free는 사용자의 GET 요청에서 Accept-Language 문자열을 캡처합니다.이 문자열은 사용자가 루트 디렉토리를 요청할 때 적절한 언어 하위 디렉토리 를 선택하는 데 사용됩니다.
